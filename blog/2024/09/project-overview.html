Three Interleaved Languages

The core of the Oils project is:

  • 2016 - OSH

    • A POSIX- and bash-compatible shell. It runs your existing shell scripts!

  • 2019 - YSH

    • A new shell with structured data. It's influenced by common languages like Python and JavaScript.

  • 2023 - J8 Notation

    • A small, optional upgrade of JSON. If we want shell programs to be simple, we should also fix the data they operate on.

You can use OSH by itself, YSH by itself, or upgrade from OSH to YSH.

Seven Parts of YSH

Let's break YSH down into 7 parts. Some of them correspond to chapters in the Oils Reference.

For more detail, see A Tour of YSH.

 

Part Notes Example

Expressions syntax like Python/JS

Conclusion

I'm glad I took the time to make these tables. What I take away is:

  • YSH is a reasonably small language, with 7 familiar parts.
  • But it's just one of 13 parts of the project, which means the project itself is big. How far we get with it depends on contributions!
