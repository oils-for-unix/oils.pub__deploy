Packaging

Summary of Oils Packaging Guidelines:

  • The oils-for-unix tarball is the fast shell in C++, completed in 2024. The distro package should be called oils-for-unix.
  • The oils-ref tarball is the reference implementation, that runs under CPython.
    • It should not be used, so there's no need to create oils-ref packages.
  • The project is now called Oils, or Oils for Unix. There is no more oil!
